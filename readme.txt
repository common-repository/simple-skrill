Simple Skrill / Moneybookers plugin

== Description ==

Skrill (Moneybookers) is one of the world's largest online payment providers currently being used by over 135,000 merchants. 
Their payment network provides more than 100 payment options, with 41 currencies covering 200 countries and territories, through just one partner. 

With one easy connection you can instantly enter new markets and grow your business. 

Join 30.7 satisfied million members...and more every day.

Simple Skrill is a plugin which helps you to add Buy Now or Donation button very easily via simple shortcode.

Quick usage guide:

Add this simple shortcode into your post or page like this:

[skrill merchant="test@skrill.com" language="EN" amount="5" currency="USD" description="Description:" text="Donation for WP Salad" button="Donate"]

Simple Skrill is plugin created by www.wpsalad.com

== Installation ==

1. Upload `ws-simple-skrill` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Put shortcode in any post or page you want and you are ready to go. Example:
[skrill merchant="support@wpsalad.com" language="EN" amount="5" currency="USD" description="Description:" text="Donation to WPSalad.com" button="Donate"]
